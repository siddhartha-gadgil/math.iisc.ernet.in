Title

: 

The Behavior of the Mordell-Weil Groups for an Elliptic Curve.
Abstract :

The set of points on an elliptic curve with coordinates in a given field K can be naturally made into an abelian group. The structure of this group varies considerably as the field K is allowed to vary. Nevertheless there is some degree of regularity to this behavior, partly proven, but mostly conjectural. We will describe what we know about this topic.
