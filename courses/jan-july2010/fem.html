Mathematical Theory of Finite Element Methods

Basic concepts in Sobolev spaces : Basic elements, Approximation by smooth functions, Extension theorem, Poincare in equality, Trace theorem and Green's theorem. Finite Elements : Weak solutions of
Elliptic boundary value problems, Lax-Milgram lemma. Galerkin formulation and Cea's lemma, Construction of finite element spaces, Polynomial Approximation and Interpolation theory. Convergence Analysis :
Error estimates and Aubin-Nitsche duality argument. Non-conforming finite elements, A posteriori error estimates, Numerical integration, Computation of finite element solutions. MATLAB implementation of
finite element methods.
